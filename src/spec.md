# Specification

## Summary
**Goal:** Add an AI chatbot assistant named "Vedu" to help users learn about Vedahealar services and book appointments through conversational interaction.

**Planned changes:**
- Add floating chat bubble icon in bottom-left corner that opens a chat interface
- Implement chat UI with message history, user input field, and distinct styling for user/bot messages
- Create backend chat endpoint to provide contextual information about Vedahealar services (numerology, name correction, lucky numbers, reiki healing, business name analysis)
- Enable Vedu to guide users through appointment booking by conversationally collecting name, email, phone, date, time, and service selection
- Submit completed bookings through the existing Google Apps Script endpoint
- Display automatic greeting message when chat is first opened

**User-visible outcome:** Users can interact with the Vedu chatbot assistant to ask questions about Vedahealar services and book appointments through a conversational interface, with the chat bubble accessible from the bottom-left corner of the page.
